Account Roles

Craft tailored permission sets for each role so teammates only see the menus they need.

Tip: Builder has full access automatically.

How permissions work

Each menu path represents a screen inside FarmVista. Toggle the abilities a role should have. Unchecked paths are hidden from that role.

  • View lets them open the page and see data.
  • Create allows adding new records or entries.
  • Edit grants updating existing information.
  • Delete removes records permanently.
  • Archive moves content out of active workflows.
